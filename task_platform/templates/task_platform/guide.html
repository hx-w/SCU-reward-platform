1. 注册与登录

1.1 注册

  1. 点击主页右上角按钮进入注册界面
  2. 根据注册页面指引填写个人信息。用户名一旦确认不可更改
    注意：必须使用学生邮箱进行注册，以保证四川大学学生身份信息真实。
  3. 填写完毕后点击按钮，并登录学生邮箱验收认证邮件并点击链接完成验证

1.2 登录

  1. 注册邮箱验证成功后才可以进行登录操作
  2. 正确填写用户名、密码和验证码后点击按钮进行登录，将自动跳转至主页，可以看到登录按钮显示称为用户名，表示登录成功

2. 查看、修改个人信息

  1. 登录后，点击主页上方显示用户名的按钮（原本是登录按钮）进入个人主页，可查看账号现有赏金以及与自己有关的各种状态的任务
  2. 点击主页上方按钮，网站右侧弹出个人信息界面
  3. 根据页面指引更改需要修改的个人信息，完毕后点击按钮即可保存
  4. 点击保存按钮或页面其他位置均可退出修改信息界面

3. 发布任务

  1. 登录后点击主页右上角按钮进入发布界面
  2. 根据页面指引填写发布任务所需信息(为任务信息完整，请尽可能多的描述清晰！)，点击选择赏金模式或猎人模式
  3. 点击按钮，创建任务
    注意：发布任务需要缴纳5元押金，若账号目前赏金不足将进入充值页面进行充值
  4. 若任务超时，/////////////////////////////////////////////////////////////////////////////////////////////

3.1 开始任务

  1. 从主页或个人页面的“未开始任务”分类中找到想要开始的任务，点击标题进入任务详情
  2. 详情页下方显示现有投标用户以及竞争价位。选择自己最心动的猎人，框选后点击按钮，确定即可开始任务

3.2 完成或撤销任务

  1. 从主页或个人页面的“进行中任务”分类中找到想要完成或撤销的任务，点击标题进入任务详情
  2. 点击按钮即可完成任务，押金与对应任务用户完成交易
  3. 点击按钮即可撤销任务，因你方中断进行中任务，押金一部分将作为惩罚金不被退回

4. 投标接取任务

4.1 接取任务

  1. 从主页或个人页面的“进行中任务”分类中找到想要完成或撤销的任务，点击标题进入任务详情
  2. 如果想接受这个任务，在页面下方根据提示提出并填写自己的报价。 注意：提交报价时你需要支付与你的报价相同金额的赏金币作为押金。
  3. 填写完成后，点击按钮进行提交。
  4. 页面显示“提交成功”字样，表示提交成功。期待自己提供的价位和描述可以使任务发布者心动吧！

4.2 主动终止任务

  1. 从主页或个人页面的“进行中任务”分类中找到想要中止的任务，点击标题进入任务详情
  2. 在页面下方点击按钮进行任务中止。

5. 基于任务的聊天

  1. 从个人页面的任务列表中找到想要与对方进行聊天任务，点击“聊天室”下的字母图标进入聊天室界面
  2. 在聊天室界面左侧显示与你相关的所有任务，点击即可切换至相应任务与其发布者/接收者进行聊天
  3. 在界面右下角聊天框输入想要发送的内容，点击右下角按钮即可发送
  4. 注意：相关任务结束后便不可以继续聊天了哦！

6. 赏金的充值、提现

6.1 赏金的充值

  1. 登录后，在个人页面的右上方点击按钮进入充值界面
  2. 填写需要充值的赏金数量后（与RMB1：1，单位为元），点击按钮跳转至支付界面
  3. 使用支付宝扫码或网页支付，成功后可以返回至个人页面查看赏金余额
